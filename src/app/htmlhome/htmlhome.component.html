Wat is HTML?

  • HTML staat voor HYPER TEXT MARKUP LANGUAGE
  • HTML is de standaard voor het maken van web pagina's
  • HTML beschrijft de structuur van een webpagina
  • HTML bestaat uit een serie van elementen
  • HTML elementen beschrijven hoe de pagina eruit moet zien aan de browser

Wat is een HTML element?

Een HTML element is gedefiniëerd door een begin tag, content en een eind tag.

Enkele veel gebruikte elementen

Element Betekenis
<p> Paragraaf
<br> Nieuwe lijn
<hr> Lijn tekenen
<b> Vetgedrukt
<i> Schuingedrukt
<sup> superscript
<sub> subscript
<!-- en --> Commentaar
